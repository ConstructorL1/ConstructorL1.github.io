# 计价规范

  • 工程量清单计价是一种主要由市场定价的计价模式。
    • 使用国有资金投资的建设工程发承包，必须采用工程量清单计价；
    • 非国有资金投资的建设工程，宜采用工程量清单计价;

# 作用

工程量清单是工程量清单计价的基础，

  • 贯穿于建设工程的招投标阶段和施工阶段，
  • 是编制招标控制价、投标报价、计算工程量、支付工程款、调整合同价款、办理竣工结算以及工程索赔等的依据。
  • （1）为投标人的投标竞争提供了一个平等和共同的基础
  • （2）是建设工程计价的依据
  • （3）是工程付款和结算的依据
  • （4）是调整工程价款、处理工程索赔的依据

# 编制

# 提供方

  • 招标工程量清单必须作为招标文件的组成部分，由招标人提供, 并对其准确性和完整性负责。2014.30

# 编制方

  • 招标工程量清单应由具有编制能力的招标人或受其委托具有相应资质的工程造价咨询人进行编制。

# 编制单位

招标工程量清单应以**单位（项）**工程为单位编制。

# 编制依据

2017.65

  • (1) 《建设工程工程量淸单计价规范》 GB 50500—2013和相关工程的国家计量规范
  • (2) 国家或省级、行业建设主管部门颁发的计价定额和办法
  • (3) 建设工程设计文件及相关材料
  • (4) 与建设工程有关的标准、规范、技术资料
  • (5) 拟定的招标文件
  • (6) 施工现场情况、地勘水文资料、工程特点及常规施工方案
  • (7) 其他相关资料

# 组成部分

应由分部分项工程量清单，措施项目清单，其他项目清单，规费和税金项目清单组成。 【分措他规税】

# 分部分项工程量清单

  • 分部分项工程量清单所反映的是拟建工程分部分项工程项目名称和相应数量的明细清单，招标人负责编制。
  • 包括项目编码、项目名称、项目特征、计量单位、工程量和工作内容。

# 项目编码

2017.5

  • 项目编码是分部分项工程和措施项目清单名称的阿拉伯数字标识。分部分项工程量清单项目编码分五级设置，12位阿拉伯数字表示。 【加砖不想担】（家、专、部、项、单）
  • 1-2位：相关工程国家计量规范代码
  • 3-4位：专业工程顺序码
  • 5-6位：分部工程顺序码
  • 7-9位：分项工程项目名称顺序码
  • 10〜12位：清单项目编码。
  • 1-9位按各专业计量规范设置；10-12位清单项目编码应根据拟建工程的工程量清单项目名称设置，同一招标工程编码不得有重码 ，由招标人针对招标工程项目具体编制，并应自001起顺序编制。

# 项目名称

  • 应根据《计量规范》的项目名称结合拟建工程的实际确定。
  • 一般以工程实体命名。
  • 编制工程量清单时，应以附录中的项目名称为基础，考虑该项目的规格、型号、材质等特征要求，并结合拟建工程的实际情况，对其进行适当的调整或细化，使其能够反映影响工程造价的主要因素。

如《房屋建筑与装饰工程工程量计算规范》GB 50854-2013 中编号为“010502001” 的项目名称为“矩形柱”，可根据拟建工程的实际情况写成“C30 现浇混凝土矩形柱 400×400”。

# 项目特征

  • 清单项目特征的描述应根据现行计量规范附录中有关项目特征的要求，结合技术规范、标准图集、施工图纸，按照工程结构、使用材质及规格或安装位置等，予以详细而准确的表述和说明。
  • 清单项目特征主要涉及项目的自身特征(材质、型号、规格、品牌)、项目的工艺特征以及对项目施工方法可能产生影响的特征。
# 项目特征作用
  • （1）项目特征是区分清单项目的依据。
  • （2）项目特征是确定综合单价的前提。
  • （3）项目特征是履行合同义务的基础。
